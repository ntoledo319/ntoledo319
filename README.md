## Current focus: EOLkits

**Amazon Linux 2 reaches end of support on June 30, 2026.** After that, no more security patches for anything built on it.

- [**EOLkits**](https://github.com/ntoledo319/EOLkits) — free, MIT-licensed CLIs for AWS runtime deprecations. One CLI per deadline: scan, codemod, IaC patching, staged canary deploys, rollback. Works offline via fixtures.
- [Instant AL2 compatibility checker](https://ntoledo319.github.io/amazon-linux-2-eol/) — paste a Dockerfile or `/etc/os-release` in your browser, get flagged patterns and fixes. Nothing leaves your machine.
- [eolkits.com](https://eolkits.com) — deadline status board, [calendar feed of AWS deprecations](https://eolkits.com/deprecations.ics), and a [done-for-you migration audit](https://eolkits.com/audit) if you'd rather hand it off.
